# Function generateInsertSQL

## Syntax

```js
dscopy.generateInsertSQL(CopyContextObject)
```

### Parameters

Parameter | Type | Description
--------- | ---- | -----------
`CopyContextObject` | any | what containts metadata of data to be copied

### Returns

Type | Description
---- | -----------
String | Sum of `x` and `y`


## Examples
